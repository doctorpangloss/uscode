uscode
======

United States Code

To run the scripts on Mac:

    # Install brew
    ruby -e "$(curl -fsSL https://raw.github.com/mxcl/homebrew/go)"
    # Install npm
    brew install npm
    # Install git
    brew install git
    # Install meteor
    curl https://install.meteor.com | /bin/sh
    # Install meteorite
    sudo -H npm install -g meteorite
    # Run and open the web app
    cd www
    mrt & open http://localhost:3000
